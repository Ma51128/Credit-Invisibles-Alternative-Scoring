# Credit Invisibles — Alternative Credit Scoring Dashboard

## Project Overview

This project develops an Alternative Credit Scoring Framework for individuals with limited or no traditional credit history.

The scoring model uses:

- Transaction behavior
- Recharge patterns
- Utility bill payments
- Savings habits
- Digital engagement metrics

to estimate customer risk profiles and financial trustworthiness.
